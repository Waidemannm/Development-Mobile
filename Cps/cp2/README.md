# 📱 Cadastro & Perfil App

Aplicativo mobile desenvolvido com Expo e Expo Router, com foco em cadastro de usuário, validação de dados, uso de máscaras e persistência local.

---

## 🚀 Funcionalidades

- Cadastro de usuário  
- Validação de formulário  
- Máscaras para CPF e telefone  
- Navegação entre telas  
- Persistência de dados com AsyncStorage  
- Autopreenchimento automático ao abrir o app  

---

## 🧭 Telas

- **Cadastro** → formulário de entrada de dados  
- **Perfil** → exibição das informações do usuário  

---

## 📝 Campos do Formulário

- Nome  
- RM  
- Telefone 
- CPF 

---
